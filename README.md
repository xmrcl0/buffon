Agulha de Buffon
================


Calcula o valor de PI usando a técnica de Buffon.


----

Compilação
----------

Efetuar a compilação do código-fonte:


        $ gcc buffon.c -o buffon -lm


----

Uso 
---
Executar com a opção help:


        $ ./buffon -h


----

Exemplos
--------

Lança uma agulha de tamanho 1 1000 vezes: 


        $ buffon -n 1000 -l 1
